Ко времени

КП «ОИС» совместно с УК «Суворовский» (по просьбе последней), провели рейд с целью обследования крыш домов для установки солнечных панелей или проведения ремонта.

Безумно жаркое лето закончилось и, по-написанному, «оглянуться не успела, как зима катит в глаза».

Наше дело — организовать операторов для перекладки сетей и определить бесхозные кабели для демонтажа и удаления с крыш.

Работа есть практически на каждом объекте. Для наглядности — один из типичных примеров.

Дом по ул. Семёна Палия, 96. Нарушения обычные, ничего особенного. Но мешают застеклить окно, в негерметичные отверстия по кабелям в дом затекает вода, сети не закреплены, как надо и пр.

Всё, что создавало помехи для работ на крыше, устранено. Сделана обычная и очень полезная работа. А главное — ко времени.
